## 🚀 About Me

I'm a computer science student at **FSHN (Fakulteti i Shkencave të Natyrës)** in Tirana, Albania. I build full-stack web applications end-to-end — from database design and REST APIs to modern React frontends — and I care about quality: I write automated tests and set up CI/CD pipelines so things don't break in production.

I don't just follow tutorials. Every project I build solves a real problem.

Currently working on:

- 🧠 **DocuMind AI** — Upload documents, get AI-powered summaries, compare files side by side, and chat with your PDFs
- 🎟️ **Bileta.al** — An Eventbrite-style event booking platform built specifically for Albania
- 📄 **CareerAI** — An AI-powered resume builder that tailors CVs to job descriptions automatically

## 📌 Featured Projects

### 🧠 [DocuMind AI](https://github.com/MateosXhukellari) — AI Document Intelligence Platform

> Upload PDFs, get AI-powered summaries, compare documents side by side, and chat with your files.

- **Stack:** React + TypeScript + TanStack Router/Query + Zustand + Zod · Spring Boot (Java) · Python AI Microservice · PostgreSQL
- **Features:** JWT auth · Document upload & management · AI summarization · Multi-document comparison · In-app AI chat · Dockerized with CI/CD via GitHub Actions
- **Status:** 🔨 In Development

---

### 🎟️ [Bileta.al](https://bileta-al-frontend.vercel.app/) — Albanian Event Booking Platform

> Eventbrite for Albania. Organizers list events, customers buy tickets, platform takes commission automatically.

- **Stack:** React + TypeScript · Spring Boot · PostgreSQL · Stripe Connect
- **Features:** Multi-role auth (Customer / Organizer / Admin) · QR code tickets · Stripe payment splitting · Email confirmations via SendGrid
- **Status:** 🔨 In Development

---

### 📄 [CareerAI](https://github.com/MateosXhukellari/cvcraft) — AI Resume Builder

> Upload your master CV and a job description — AI tailors your resume automatically.

- **Stack:** React + TypeScript · Spring Boot · Python AI Microservice · Electron
- **Features:** AI CV customization · Application tracker · Job management · Kanban board
- **Status:** 🔨 In Development

---

### 🏥 [HealthCare+](https://health-care-frontend-pied.vercel.app/) — Medical Appointment Booking
